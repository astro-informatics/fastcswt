Compute number of effective coefficients in an extended coefficient exclusion mask and write results to either the standard output or to a file.

Usage: cswt_mask_nonzero

  • [-help]: Display usage information.
  • [-inp filename_mask]: Name of file containing extended coefficient
  • [-out filename_out]: Name of file to write results to. If not specified then results are written to the screen.
