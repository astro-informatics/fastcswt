Invert a coefficient mask by converting ones to zeros, and zeros to ones.

Notes:

  • Original mask should be in one/zero format, not display format.

Usage: cswt_mask_invert
  • [-help]: Display usage information.
  • [-inp filename_inp]: Name of input file containing original mask to be inverted.
  • [-out filename_out]: Name of output file created that will contain the inverted mask.
